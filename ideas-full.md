## 一、核心价值主张

**"图片不出本地，洞察来自云端"**

ViT Secure Inference 是一套**端云协同隐私推理**方案：用户的图片/文档在本地设备上通过轻量 ViT 编码器提取特征向量，再将压缩后的向量上传云端（GLM-4.5V）完成语义理解。全程**原始像素数据不离开本地**，同时保留云端大模型的完整推理能力。

### 隐私保护等级

| 场景 | 传统云端推理 | ViT Secure Inference |
|------|-------------|----------------------|
| 医疗影像（X光/CT） | 原始影像上云 ⚠️ | 仅特征向量上云 ✅ |
| 企业文件（合同/图纸） | 原始文档上云 ⚠️ | 仅嵌入向量上云 ✅ |
| 个人照片/证件 | 原始图片上云 ⚠️ | 仅特征上云 ✅ |
| 数据合规（PIPL/GDPR） | 需要额外脱敏 ⚠️ | 架构级合规 ✅ |

## 二、技术架构

### 数据流

```
用户图片（本地）
    │
    ▼
本地 ViT 编码器（~300M 参数，macOS 本地运行）
• 模型：mmproj-GLM-4.5V-Q8_0.gguf
• 运行时：Metal GPU 加速（Apple Silicon）
• 输出：压缩特征向量（~3x 压缩率）
    │
    ▼  仅特征向量（无原始像素）
    │
GLM-4.5V API（智谱 AI 云端）
• 接收特征向量
• 完成语义推理
• 返回分析结果
    │
    ▼
用户获取分析结果
```

## 四、竞争分析

| 方案 | 隐私保护 | 推理质量 | 本地计算要求 | 备注 |
|------|---------|---------|------------|------|
| 纯云端 API（OpenAI/GLM）| ❌ 原始数据上云 | ⭐⭐⭐⭐⭐ | 无 | 主流方案 |
| 本地部署大模型（LLaVA等）| ✅ 完全本地 | ⭐⭐⭐ | 极高（GPU 8GB+）| 质量受限 |
| **ViT Secure Inference** | ✅ 特征向量 | ⭐⭐⭐⭐⭐ | 低（CPU/Metal）| **本方案** |
| 联邦学习方案 | ✅ | ⭐⭐⭐⭐ | 极高 | 工程复杂度高 |
| TEE/安全飞地方案 | ✅✅ | ⭐⭐⭐⭐ | 需特殊硬件 | 成本高 |

**核心差异化**：以极低的本地算力代价（300M 参数 ViT vs 7B+ 全模型），换取接近云端大模型的推理质量，同时实现架构级隐私保护。
